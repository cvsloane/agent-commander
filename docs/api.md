# API Reference (Summary)

A full OpenAPI/AsyncAPI reference is planned. Key endpoints include:

## Sessions
- `GET /v1/sessions`
- `GET /v1/sessions/:id`
- `POST /v1/sessions/:id/commands`

## Approvals
- `GET /v1/approvals`
- `POST /v1/approvals/:id/decide`

## Hosts
- `GET /v1/hosts`
- `POST /v1/hosts`

## Notifications
- `POST /v1/notifications/test`
