# Alerts & Notifications

Agent Commander supports alerts through:
- Browser notifications
- Audio alerts
- In-app toasts
- Clawdbot (optional)

## Event Filters
Each channel can be filtered by event type:
- approvals
- approval_decisions
- waiting_input
- waiting_approval
- error
- snapshot_action
- usage_thresholds

## Provider Filters
Each channel can be filtered by provider (claude_code, codex, gemini_cli, shell, etc.).

## Usage Thresholds
Usage thresholds trigger alerts at 50%, 75%, 90%, and 100% by default.
