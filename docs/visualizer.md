# Visualizer

The visualizer is a full screen activity view that turns your session list into a live control room.

![Visualizer](images/visualizer.png)

## Themes

- Botspace
- Civilization
- Bridge Control

Themes are selectable in Settings. The visualizer uses the same live data feeds as the dashboard.

## What it shows

- Live session activity and attention states
- Recent events and approvals
- Quick command bar for interacting with sessions

## Access

- Navigate to `/visualizer` in the dashboard.
- Use the sidebar toggle if enabled.

## Notes

The visualizer is designed for ambient monitoring. For detailed interaction, open the session detail or console view.
