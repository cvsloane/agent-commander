# Agent Commander

**Tmux-first mission control for live agents, memory, and autonomy.**

Agent Commander discovers tmux sessions across your hosts, streams consoles live
into the browser, adds repo and global memory to every session, and gives you
autonomous orchestration without leaving the normal session runtime.

**New in v0.2.0**

- `/tmux` for windows-first pane management
- `/automation` for orchestrators, workers, governance, and work queues
- `/memory` for repo/global memory and procedural knowledge
- Hermes integration for external wakes and watchdog summaries
