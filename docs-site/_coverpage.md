# Agent Commander

**Mission control for AI agent sessions.**

Agent Commander discovers tmux sessions across your hosts, streams consoles live
into the browser, and gives you an orchestrated view of approvals, errors, and
activity. It is designed for serious multi-agent operations, not toy demos.
